SWAP
Game Design & Development

SWAP is a simple yet brutally difficult rhythm game developed by indie studio Sumi Labs. It features an amazing soundtrack created by Dance With the Dead. It was released on Steam, iOS and Google Play.

Team Size: 2
Engine: Unity

Notable Contributions:
- Full controller support including level editor.
- Settings menu and functionality.
- Jukebox and monetization pages.

Chess Attack
Game Design, Development, Art, Publishing

Chess Attack is a chess-based endless survival game where you play as the king. It was a project I created for fun while in college and was released on iOS and Android but has since moved to itch.io.

Team Size: 1
Engine: Unity

Notable Contributions:
- Game design and gameplay programming.
- Publishing on iOS Appstore and Google Play.
- Online leaderboards for both game modes.

Devour the Tower
Game Design, Development, Monetization

Devour the Tower is a casual game about a baby eating ice cream. It was created for a college senior design competition where it was awarded 1st place in the Computer Science division. The game was released on iOS and Android but has since moved to itch.io.

Team Size: 7
Engine: Unity

Notable Contributions:
- Gameplay programming.
- Rewarded ads via Unity Ads system.
- Game Center and Google Play Leaderboards integration.

Scuba Sim
Game Design, Development, Art

Scuba Sim is a game where you dodge dangerous sea creatures using your mouse. It was my first experience with game development. The original version was made with block-based programming using MIT App Inventor 2. I recreated it using Javascript in order to host it on the web, where it lives to this day.

Team Size: 1
Language: Javascript
